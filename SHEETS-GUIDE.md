# Digital Bank Savings Tracker — Sheet-by-Sheet Guide

This guide explains what each of the five tabs is for, what's on it, and how to
use it. Tabs (left to right): **Dashboard · Deposit · Transactions · Balance ·
Interest**.

**Only two tabs need your input:** the **Deposit** tab and the Transaction Log on
the **Transactions** tab. Everything else calculates automatically.

> Tip: open the file, click **Enable Editing** (yellow bar), then press **F9** so
> the date-based interest refreshes to today.

---

## 🏠 Dashboard
**Purpose:** a one-page overview of everything — read-only.

**What's on it**
- **🔑 Key Metrics**
  - *Total Current Balance (incl. interest)* — all accounts' balances added up (interest included).
  - *Total Deposited (All-Time)* — everything you've ever deposited.
  - *Interest Earned to Date (auto)* — total interest accrued so far.
  - *Total Withdrawn* — sum of your "Withdrawal" transactions.
  - A live **"Figures as of [date]"** stamp shows the day the numbers were last calculated.
- **🎯 Personal Goal Tracker**
  - *Your Savings Goal Target* — the one cell you can type in (highlighted yellow).
  - *Maya Personal Goal — Current Balance* — pulled automatically.
  - *Progress Toward Goal* — a % with a green progress bar.
- **🏦 Balance by Bank** — total balance per bank (Maya, UNO, Tonik, Banko, CIMB, GoTyme, Maribank) + grand total.
- **Charts**
  - 📈 *Savings Growth by Bank* — one line per bank over 24 months.
  - 🥧 *Savings by Bank* — share of your money per bank.
  - 📊 *Monthly Interest Earned by Bank* — stacked columns per month.
- **🗂 Chart Data** (further down) — the auto-calculated tables that feed the charts. Do not edit.

**How to use:** just read it. The only editable cell is your **Goal Target**.

## ➕ Deposit
**Purpose:** log every deposit (money you put in). This is a main input tab.

**Columns:** `Date | Bank | Account | Amount | Notes`

**How to use**
- Type each deposit on the next empty row of the table.
- **Bank** and **Account** are **drop-downs** — pick from the list so the entry
  matches an account exactly (typing a name incorrectly would break the totals).
- Use the **filter arrows** in the header to search or sort.
- The table auto-extends as you add rows.

Every deposit here flows into the account's **Principal** and **Current Balance**
on the Balance sheet automatically.

---

## 🔄 Transactions
**Purpose:** two things stacked on one tab.

### Top — 🪙 Interest Earned per Day *(auto-posted — do not edit)*
- One read-only line per account showing that account's **interest earned per
  day** (`Balance × annual rate ÷ 365`), plus a **Total per day**.
- Generated by formula; it updates when the file recalculates.
- Columns: `Date | Type | Bank | Account | Amount | Notes`.

### Below — 🔄 Transaction Log *(your input)*
- Log money you move that isn't a deposit: **withdrawals, transfers,
  adjustments, fees**. Use **negative amounts for money out**.
- Columns: `Date | Type | Bank | Account | Amount | Notes`.
- **Type / Bank / Account** are drop-downs; use the **filters** to search.
- The log sits at the bottom with nothing beneath it, so it **grows freely** as
  you add rows — no cleanup needed.

> Don't record interest here manually — it's auto-posted at the top and already
> included in your balance (recording it again would double-count).

## 💼 Balance
**Purpose:** the current balance of every account — read-only.

**Columns**
| Column | Meaning |
|--------|---------|
| Bank / Account / Account Label | which account the row is |
| **Deposits** | total deposits for this account (from the Deposit tab) |
| **Net Transactions** | sum of your Transaction Log entries (withdrawals subtract) |
| **Principal** | Deposits + Net Transactions (the money you moved) |
| **Interest / Day** | interest this account earns per day |
| **Current Balance** | Principal + interest accrued to date — **this rises a little every day** |
| **Interest (1 Yr)** | projected interest if held one year |
| **Proj. Balance (1 Yr)** | projected balance one year out |

**How to use:** read only. The **TOTAL** row sums each column. *Current Balance*
is the number that matters day to day; it already includes earned interest.

## 📈 Interest
**Purpose:** the interest engine — rates, per-day/per-month earnings, and the
live daily accrual. Mostly read-only; the **Interest Start** date is editable.

**Visible columns**
| Column | Meaning |
|--------|---------|
| Bank / Account / Account Type / Account Label | which account |
| **Annual Rate** | the yearly interest rate (Maya Savings = 10%) |
| **Frequency** | Daily / Monthly (reference for the 1-year projection & charts) |
| **Current Balance** | the account's principal (from the Balance tab) |
| **Interest / Day (est.)** | Balance × rate ÷ 365 |
| **Interest / Month (est.)** | Balance × rate ÷ 12 |
| **Proj. Balance (1 Yr)** | balance projected one year out |
| **Interest (1 Yr)** | projected one-year interest |
| **Interest Start** *(yellow, editable)* | the date interest starts accruing (pre-filled with the first deposit date) |
| **Interest to Date** | interest actually earned so far, **accrued daily** |
| **Balance Today** | Current Balance + Interest to Date |

**Hidden helper columns** (calculated in the background, hidden so the sheet fits
the screen): *Periodic Rate, Monthly Factor, Days Elapsed, Periods / Yr*. You can
unhide them via Home → Format → Hide & Unhide if you're curious.

**How to use**
- To change a rate, edit **Annual Rate**.
- To change when interest starts for an account, edit its **Interest Start**
  date (yellow).
- Everything else auto-calculates. **Interest to Date** and **Balance Today**
  grow every day because they use `TODAY()`.

## How it all connects

```
Deposit  ─┐
          ├─►  Balance (Principal)  ─►  Interest (accrues daily)  ─►  Dashboard
Transactions ┘                          │
                                        └─►  auto-posted on the Transactions tab
```

- You enter **Deposits** and **Transactions**.
- **Balance** totals them into **Principal**.
- **Interest** accrues daily on the principal and feeds **Current Balance**.
- **Dashboard** summarizes everything with metrics and charts.

No circular references, no macros — it's all standard Excel formulas.
